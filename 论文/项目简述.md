# 论文背景说明（BLE Channel Sounding 多设备测距）

本论文研究对象为基于 Bluetooth Channel Sounding 技术的室内测距方法，重点关注多设备并发测距场景下的信号处理与距离估计问题。

---

# 一、研究背景

随着物联网（IoT）技术的发展，大量终端设备需要在室内环境中实现位置感知能力。相比室外环境，室内定位无法依赖GNSS系统，因此需要基于无线通信信号实现测距与定位。

目前常见的室内定位技术包括：

- WiFi定位（基于RSSI或指纹）
- UWB定位（基于ToF）
- BLE定位（低功耗、低成本）

其中，BLE由于广泛部署于智能手机和IoT设备中，在工程上具有较高的应用价值。然而，传统BLE定位主要依赖RSSI，其测距精度较低且受环境影响较大。

为解决该问题，Bluetooth 6.0标准引入了Channel Sounding（CS）技术，使BLE设备能够通过相位和时间信息实现高精度测距。

---

# 二、Channel Sounding测距原理

BLE Channel Sounding通过设备间的双向交互测量无线信道特性。

系统中包含两个角色：

- Initiator（主设备）：发起测距过程
- Reflector（从设备）：接收并响应信号

测距过程中，系统获取两类数据：

- local IQ：主设备接收到的复数信号
- peer IQ：从设备测得并返回的信号

通过将local IQ与对应设备的peer IQ进行复数运算，可以消除设备本振引入的初始相位误差，从而获得仅与传播路径相关的相位信息。

---

# 三、问题描述：多设备并发测距

在实际应用中，系统通常需要同时对多个设备进行测距。

当Initiator发起测距时：

- 多个Reflector会同时接收信号
- 并返回各自响应信号

因此，在接收端，信号表现为多个设备信号的叠加：

r(t) = Σ A_i * exp(jφ_i)

其中：

- 每个φ_i对应不同设备的传播距离
- 信号在单频条件下无法分离

这导致：

- 无法直接从接收信号中区分不同设备
- 无法获得单个设备的距离

---

# 四、核心思路：多频相位一致性

BLE Channel Sounding具有一个关键特性：

👉 支持多频测量（跳频机制）

对于目标设备：

φ(f_k) = 2π f_k d / c

即：

- 相位随频率呈线性变化
- 在多频维度上具有一致性（coherent）

对于非目标设备：

- 由于peer IQ不匹配
- 初始相位无法消除
- 相位在不同频率上呈随机变化

👉 表现为“非相干信号”

---

# 五、算法核心思想

基于上述特性，可以得到：

在多频域中：

- 目标设备信号 → 相干累积（增强）
- 干扰设备信号 → 非相干叠加（抑制）

因此，可以通过：

👉 距离域搜索 / 相位匹配

来实现目标设备的距离估计。
