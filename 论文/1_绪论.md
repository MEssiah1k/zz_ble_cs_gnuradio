# 第1章 绪论

## 1.1 研究背景与意义

随着物联网、智能制造和智慧城市等应用的快速发展，室内环境中的高精度位置感知需求日益突出。与室外场景不同，室内定位难以直接依赖 GNSS 等卫星导航系统，因此需要借助无线通信信号完成设备间距离估计与位置感知。如何在低功耗、低成本和高精度之间取得平衡，已经成为无线定位领域的重要研究方向。

BLE 由于具备低功耗、低成本和广泛部署等优势，长期以来是室内定位与近距离测距的重要技术基础。传统 BLE 测距大多依赖 RSSI 或指纹方法，但这类方法对环境遮挡、多径衰落和设备差异较为敏感，测距精度通常难以达到高精度应用要求。为提升测距能力，Bluetooth 6.0 引入了 Channel Sounding 技术，使蓝牙设备能够通过相位和时间信息开展高精度测距研究。

然而，现有 Channel Sounding 协议主要面向点对点测距模式，即一个 Initiator 与一个 Reflector 之间进行交互。当系统需要同时对多个设备进行测距时，传统分时测距会带来更高的系统时延和更复杂的调度开销；同时，多设备响应在接收端叠加后，还会使相位观测结果更难分离。因此，研究面向多设备并发场景的 BLE Channel Sounding 测距方法，不仅具有明确的理论意义，也具有较强的工程应用价值。

## 1.2 国内外研究现状

围绕室内定位与高精度测距问题，学术界和工业界已经提出了多种技术路线。RSSI、UWB、WiFi 指纹和基于相位的测距方法是当前常见的研究方向。其中，UWB 在精度上具有优势，但部署成本和系统复杂度相对较高；WiFi 和 RSSI 方案易于部署，但受环境变化影响较大；基于相位的测距方法则在较低带宽条件下具备更高的分辨率，适合与 BLE 场景结合。

随着蓝牙标准的演进，Channel Sounding 逐渐成为研究热点。现有研究主要聚焦于单设备场景下的 PBR、RTT 和相位补偿机制，而对于多设备并发测距、目标分离和离线配对处理的研究相对有限。尤其是在真实 SDR 平台和 USRP 硬件上，如何构造稳定的多频观测、如何处理随机初相和硬件偏置、如何在叠加信号中恢复目标设备的距离信息，仍然缺乏统一且可工程落地的方法。

## 1.3 本文研究内容

针对上述问题，本文围绕 BLE Channel Sounding 多设备并发测距展开研究，主要完成以下工作。

首先，建立多设备场景下的信号与测距问题模型，分析目标设备与非目标设备在多频观测中的相位差异，为后续方法设计提供理论基础。

其次，提出一种基于多频相位一致性的距离估计方法。该方法利用 local IQ 与 peer IQ 的配对关系构造目标相干项，并通过距离搜索在频率维度上累积目标设备的相位一致性，从而实现多设备条件下的目标提取与距离估计。

再次，基于 Python 仿真工程、GNU Radio 平台和 USRP 真实数据采集链路，对所提方法进行三级验证，逐步考察其在理想信道、噪声信道、多设备叠加和真实硬件环境中的表现。

最后，构建真实数据离线处理流程，对 1to1 基线、random phase 验证和 1to2 合成数据进行统一分析，并从误差分布、距离组合和场景差异三个角度总结方法的性能特点。

## 1.4 论文结构安排

全文共分为五章，各章内容安排如下。

第1章为绪论，介绍研究背景、国内外研究现状、本文研究内容与论文结构。

第2章介绍 BLE Channel Sounding 的基本原理、RTT 与 PBR 的测距机理、多频相位一致性思想以及误差来源分析。

第3章围绕多设备并发测距问题建立系统模型，并提出基于多频相位一致性的距离估计方法。

第4章给出 Python 仿真验证、GNU Radio 系统实现、USRP 实验平台以及真实数据分析结果。

第5章对全文工作进行总结，并讨论后续可进一步改进的方向。

## 1.5 本章小结

本章介绍了多设备并发测距研究的背景与意义，梳理了与 BLE Channel Sounding 相关的研究基础，并明确了本文的主要研究内容与论文结构。通过本章的说明可以看出，本文的核心目标是在真实可实现的平台上，探索一种适用于多设备场景的高精度 BLE 测距方法。
