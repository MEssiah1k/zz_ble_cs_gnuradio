
# 第2章 BLE Channel Sounding测距原理

## 2.1 BLE Channel Sounding测距原理概述

随着室内定位应用对测距精度要求的不断提高，传统基于接收信号强度指示（Received Signal Strength Indicator，RSSI）的蓝牙测距方法已难以满足高精度定位需求。RSSI 方法实现简单、部署成本低，但信号强度容易受到遮挡、多径传播以及环境变化等因素影响，因此其测距精度和稳定性通常较差。为提升蓝牙设备之间的距离感知能力，Bluetooth Core Specification 6.0 引入了 Channel Sounding（CS）功能，用于支持更高精度的设备间测距。Bluetooth SIG 官方资料表明，Channel Sounding 的目标是为蓝牙设备提供基于信号相位与时延特征的精细测距能力，并可与数字钥匙、设备查找等应用场景结合使用。 ([Bluetooth® Technology Website](https://www.bluetooth.com/core-specification-6-feature-overview/?utm_source=chatgpt.com "Bluetooth® Core 6.0 feature overview"))

BLE Channel Sounding 的基本思想是：在两个已建立连接的蓝牙设备之间，通过标准化的测距会话获取无线信道的相位信息和传播时间信息，再结合相应的距离估计方法求取设备间距离。测距会话通常由两个角色构成，其中 Initiator 负责发起测距流程并控制交互过程，Reflector 在接收到测距信号后执行响应并返回相关测量结果。与传统 RSSI 方法不同，Channel Sounding 不直接利用接收功率推断距离，而是通过更直接地刻画无线信道传播特性来估计传播路径长度，因此在理论上能够获得更高精度和更强鲁棒性。 ([Bluetooth® Technology Website](https://www.bluetooth.com/bluetooth-resources/bluetooth-channel-sounding-a-technical-overview/?utm_source=chatgpt.com "Bluetooth® Channel Sounding: A Technical Overview"))

从物理层角度看，Channel Sounding 在 2.4 GHz ISM 频段中定义了专用的 CS 信道索引，并支持跨多个频点执行测量过程。Bluetooth SIG 官方规范说明，CS 定义了 79 个 RF 信道索引，其中部分信道可用于 CS 通信。多信道测量机制是 Channel Sounding 区别于传统单信道或仅依赖 RSSI 测距方法的重要特点之一。通过跨多个频率点重复测量，可以在一定程度上降低随机误差影响，并为后续相位解模糊提供条件。([Bluetooth® Technology Website](https://www.bluetooth.com/wp-content/uploads/Files/Specification/HTML/Core-60/out/en/low-energy-controller/channel-sounding.html?utm_source=chatgpt.com "Part H Channel Sounding"))

总体而言，BLE Channel Sounding 的理论基础可归纳为两个方面：一是基于信号往返传播时间的测距方法，即 Round-Trip Time（RTT）；二是基于信号传播相位变化的测距方法，即 Phase-Based Ranging（PBR）。其中，RTT 更侧重于利用时间信息实现距离估计，并可作为安全距离约束的辅助手段；PBR 则通过多频相位测量实现高精度距离估计，是 Channel Sounding 中实现精细测距的核心机制。 ([Bluetooth® Technology Website](https://www.bluetooth.com/learn-about-bluetooth/feature-enhancements/channel-sounding/?utm_source=chatgpt.com "Bluetooth® Channel Sounding"))

## 2.2 RTT测距原理

Round-Trip Time（RTT）测距方法是通过测量信号从发起设备传播到响应设备，再由响应设备返回至发起设备所经历的总时间来估计设备间距离。其基本思想较为直观：若已知电磁波传播速度，则可根据传播时间反推出传播路径长度。

设信号在无线信道中的单程传播时间为 (t_p)，信号往返总传播时间为 (t_{RTT})，设备内部处理时延为 (t_{proc})，则有：

[
t_{RTT}=2t_p+t_{proc}
]

因此，单程传播时间可表示为：

[
t_p=\frac{t_{RTT}-t_{proc}}{2}
]

进一步地，若电磁波传播速度近似取为光速 (c)，则设备间距离 (d) 可表示为：

[
d=c t_p=\frac{c\left(t_{RTT}-t_{proc}\right)}{2}
]

式中，(c) 表示电磁波在自由空间中的传播速度，通常近似取 (3\times10^8 \text{ m/s})；(t_{RTT}) 为测得的往返时间；(t_{proc}) 为响应设备处理并返回信号所引入的时延。该公式表明，RTT 测距的核心在于准确测量总往返时间并有效扣除设备内部处理延迟。

RTT 方法的优点在于物理意义清晰，不依赖于相位周期信息，因此不存在相位模糊问题。此外，Bluetooth SIG 官方资料指出，RTT 在 Channel Sounding 中还可作为安全距离边界的辅助机制，用于交叉验证基于相位的测距结果，从而降低中间人转发攻击等安全风险。([Bluetooth® Technology Website](https://www.bluetooth.com/learn-about-bluetooth/feature-enhancements/channel-sounding/?utm_source=chatgpt.com "Bluetooth® Channel Sounding"))

然而，RTT 测距方法也存在明显局限。由于无线电信号传播速度极快，当距离尺度进入米级甚至分米级时，对时间测量分辨率提出了极高要求。例如，1 m 单程传播时间约为 3.33 ns，因此若要实现亚米级测距，系统需要具备纳秒量级的时间测量能力。对于低功耗蓝牙设备而言，这种高精度时间测量在实现上存在一定难度。因此，在 BLE Channel Sounding 中，RTT 通常作为补充测距机制，而非唯一的高精度距离估计手段。([Bluetooth® Technology Website](https://www.bluetooth.com/learn-about-bluetooth/feature-enhancements/channel-sounding/?utm_source=chatgpt.com "Bluetooth® Channel Sounding"))

## 2.3 基于相位的测距原理（PBR）

相比 RTT 方法，Phase-Based Ranging（PBR）更适合实现高精度距离估计。其基本原理在于：无线信号在传播过程中会产生与传播路径长度相关的相位变化，因此可以通过测量接收信号相位来反推出传播距离。

设载波频率为 (f)，波长为 (\lambda)，传播距离为 (d)，则信号传播所引起的相位变化可表示为：

[
\phi = \frac{2\pi d}{\lambda}
]

由于波长与频率之间满足关系

[
\lambda=\frac{c}{f}
]

故相位变化还可写为：

[
\phi = \frac{2\pi f d}{c}
]

上述公式表明，在理想直达传播条件下，信号相位与传播距离成线性关系。当测得相位 (\phi) 后，理论上可以反推出距离 (d)：

[
d=\frac{c\phi}{2\pi f}
]

然而，PBR 测距的关键问题在于相位具有 (2\pi) 周期性。实际测得的相位通常只能表示为模 (2\pi) 的结果，即：

[
\phi_{\text{meas}}=\phi \bmod 2\pi
]

这意味着不同的实际距离可能对应相同的测量相位。例如，当传播距离增加一个完整波长时，测得的相位值将重复出现。因此，仅依赖单一频率下的相位测量，通常无法唯一确定真实距离，这一问题即为相位模糊问题。

从工程角度看，PBR 的优势在于相位对距离变化十分敏感，因此在高信噪比和较稳定信道条件下，能够实现比 RSSI 更高的测距精度。Bluetooth SIG 公开资料也将 PBR 视为 Bluetooth Channel Sounding 实现高精度 fine ranging 的核心机制之一。([Bluetooth® Technology Website](https://www.bluetooth.com/learn-about-bluetooth/feature-enhancements/channel-sounding/?utm_source=chatgpt.com "Bluetooth® Channel Sounding"))

因此，PBR 测距的理论重点并不仅在于相位与距离的对应关系，更在于如何通过多频测量消除单频相位模糊，从而实现对实际距离的唯一估计。

## 2.4 多频相位测距原理

为解决单频 PBR 中存在的相位模糊问题，BLE Channel Sounding 采用了多频相位测量机制。其基本思想是：在多个不同载波频率上重复测量信号相位，通过分析相位随频率变化的规律来恢复真实距离。

设在两个不同频率 (f_1) 和 (f_2) 下测得的传播相位分别为 (\phi_1) 和 (\phi_2)，则有：

[
\phi_1 = \frac{2\pi f_1 d}{c}, \qquad
\phi_2 = \frac{2\pi f_2 d}{c}
]

两式相减可得相位差：

[
\Delta \phi = \phi_2 - \phi_1 = \frac{2\pi (f_2-f_1)d}{c}
]

由此可得距离表达式：

[
d = \frac{c,\Delta \phi}{2\pi (f_2-f_1)}
]

这一公式说明，当已知不同频率之间的相位差时，可以利用频率差来反推传播距离。与单频测量相比，多频测量相当于引入了额外的约束条件，从而可在更大范围内消除相位模糊。

更一般地，若在多个频率点 (f_1,f_2,\dots,f_N) 上分别进行测量，则理想情况下各频点相位应满足如下线性关系：

[
\phi_i = \frac{2\pi f_i d}{c} + \phi_0,\qquad i=1,2,\dots,N
]

其中，(\phi_0) 表示与设备本振、初始相位偏移等相关的常量项。在多频测量条件下，可以通过对 (\phi_i) 与 (f_i) 的关系进行拟合、差分或搜索，得到更稳定的距离估计结果。你开题报告中提到“通过对多个信道测量结果进行综合分析，可以减小相位测量误差并提高距离估计精度”，其理论基础正是来源于这种多频约束。

需要指出的是，Bluetooth Channel Sounding 标准本身提供的是标准化的相位和时延测量机制，而不是唯一固定的距离估计算法。Bluetooth SIG 明确说明，CS 规范并不规定单一距离算法，具体距离求解方法可以根据应用需求独立实现。也正因为如此，多频相位关系为后续算法设计提供了足够的建模空间。([Bluetooth® Technology Website](https://www.bluetooth.com/learn-about-bluetooth/feature-enhancements/channel-sounding/?utm_source=chatgpt.com "Bluetooth® Channel Sounding"))

综上，多频相位测量机制是 BLE Channel Sounding 区别于传统 RSSI 测距方法的重要特征，也是其实现高精度测距的关键理论基础。

## 2.5 测距误差来源分析

尽管 BLE Channel Sounding 在理论上能够实现高精度测距，但实际系统中的测距结果仍会受到多种误差因素影响。为后续算法设计与系统评估提供理论基础，有必要对主要误差来源进行分析。

### 2.5.1 多径效应

在室内环境中，无线信号通常不仅沿直达路径传播，还会受到墙体、地面、金属设备以及人体等反射体的影响，形成多条传播路径。设接收信号由 (L) 条路径叠加而成，则可表示为：

[
r(t)=\sum_{l=1}^{L} a_l e^{j\phi_l}
]

其中，(a_l) 和 (\phi_l) 分别表示第 (l) 条路径的幅度和相位。由于接收端实际测得的是多条路径叠加后的综合结果，因此其相位往往不再仅由直达路径距离决定。这会导致基于相位的距离估计产生偏差，尤其在非视距（NLOS）或强反射环境下更加明显。

### 2.5.2 噪声影响

接收机热噪声、量化噪声以及环境电磁干扰均会影响相位和时间测量精度。若将接收信号表示为：

[
y(t)=s(t)+n(t)
]

其中，(s(t)) 为理想接收信号，(n(t)) 为噪声项，则噪声会引起相位测量抖动，并降低多频拟合结果的稳定性。当信噪比较低时，相位测量误差会显著增大，进而影响距离估计准确性。

### 2.5.3 频率偏差与本振误差

在实际硬件系统中，不同设备的本地振荡器通常存在频率偏差和初始相位偏移。特别是在多频测量过程中，若载波频率存在偏移，则相位与频率的理论线性关系会受到破坏，从而影响距离估计结果。你开题报告中提到 Zand 等人分析了晶振偏差与噪声对测距精度的影响，这正说明频率偏差是 PBR 测距中不可忽略的误差来源之一。

从原理上看，频率偏差会使理论相位项

[
\phi = \frac{2\pi f d}{c}
]

中的 (f) 偏离理想值，导致测得的相位差与真实传播距离之间出现系统性误差。若不加处理，这类误差会随着频率和距离的变化而累积，并降低多频距离解算精度。

### 2.5.4 小结

总体而言，多径效应、噪声以及频率偏差是影响 BLE Channel Sounding 测距性能的主要因素。其中，多径效应主要破坏传播路径与相位之间的一一对应关系，噪声主要降低测量稳定性，而频率偏差则会引入系统性误差。因此，在后续算法设计和性能分析中，需要重点考虑上述因素对 RTT 和 PBR 测距结果的影响。
