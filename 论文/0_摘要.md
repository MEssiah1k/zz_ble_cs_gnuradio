# 摘要

随着蓝牙低功耗技术在物联网终端中的广泛部署，基于 Bluetooth Channel Sounding 的高精度测距逐渐成为室内定位研究的重要方向。然而，现有 Channel Sounding 机制主要面向点对点测距场景，在多设备并发条件下不仅通信时延和能耗随设备数量增加而上升，而且多个设备返回信号在接收端叠加后还会引入相位干扰，影响距离估计精度。针对上述问题，本文围绕多设备并发测距场景，构建了从理论建模、算法设计到仿真与真实数据验证的完整研究链路。
本文首先分析了 BLE Channel Sounding 的 RTT 与 PBR 测距原理，并在此基础上建立多设备信号叠加模型，进一步提出基于多频相位一致性的目标提取与距离估计方法。该方法利用 local IQ 与 peer IQ 的复数配对关系，在多频观测条件下增强目标设备的相干分量，并抑制非目标设备带来的非相干干扰，从而实现对目标设备距离信息的提取。随后，本文在 Python 仿真环境中完成了单目标、随机初相、多设备叠加、多径以及衰落条件下的验证；在 GNU Radio 与 USRP 平台上实现了多频跳频、双向配对、随机相位注入和连续 IQ 采集，并进一步构建了真实 1to2 场景下的离线处理链路。
实验结果表明，本文方法在理想单目标条件下能够正确恢复距离，3 m 单设备基线在校准相消后保持稳定的结果，引入随机相位后结果不变。该基线中的固定偏移主要来自 USRP 平台链路时延与硬件偏置，属于平台特性而非算法失配。在真实 1to2 数据上，本文最终构造了 26 组距离组合、64 个组别配置和 2656 个目标测距样本，整体距离估计结果的 MAE 为 1.085 m，中位绝对误差为 0.41 m，90% 分位绝对误差为 2.09 m。结果进一步表明，该方法在近距离和中近距离组合上表现较好，而在远距离弱目标以及远近混合场景下误差会显著增大。
综上，本文验证了基于多频相位一致性的多设备并发测距思路在 BLE Channel Sounding 场景中的可行性，并为后续提升复杂室内环境下的测距稳定性与多设备处理能力提供了实现基础。
关键词：BLE Channel Sounding；多设备并发测距；多频相位一致性；Phase-Based Ranging；USRP
